## Step 3: 전문가 검토 필요 항목

아래 항목들은 웹 검색만으로 검증이 어려워 **해당 분야 전문가 확인이 필요**하다.

| 항목 | 보고서 내용 | 전문가 확인 필요 사유 |
|------|-----------|---------------------|
| LCOE 비교 수치 | 태양광 70~90원, CCGT 100~130원 등 | 연도·조건별 편차가 크며 공식 발표 수치 확인 필요 |
| 용량시장 보상 수준 | 7~9만원/kW·년 | 전력거래소 공식 자료와 대조 필요 |
| 출력제한 횟수 | 2025년 약 120회 | 전력거래소 계통운영 실적 확인 필요 |
| 램프율 25GW | 2025년 봄철 기준 | 전력거래소 실시간 계통 데이터 확인 필요 |
| 탄소비용 영향 | 톤당 5만원 시 석탄 LCOE 40~50원 상승 | 에너지경제연구원 시나리오 분석 확인 필요 |
| 건설 비용(kW당) | SGT 40~60만원, CCGT 70~100만원 | 최근 프로젝트 실적 기반 확인 필요 |

| 구분 | 건수 |

| 🔍 전문가 확인 필요 | 6건 |

### 신뢰도 등급: **주의 필요** ⚠️

- 기술 원리, 개념 설명 등 **정성적 내용**은 대체로 정확하다.
- 구체적 **수치(출력, 효율, 혼소율, 설비용량)**에서 Hallucination이 집중적으로 발생하였다.
- 특히 **서로 다른 모델/세대의 사양을 혼동**하는 패턴이 두드러진다.
- 이 보고서를 공식 문서로 사용하기 전에 **반드시 수치 데이터를 원본 출처에서 재검증**해야 한다.

## 권장 조치

1. 보고서 내 모든 수치 데이터를 공식 출처(제조사 사양서, 정부 통계)에서 재확인
2. 두산에너빌리티 S1/S2+ 사양을 명확히 구분하여 기재
3. Siemens 수소 혼소율 75% → 50%로 수정
4. Mitsubishi Power 가스터빈 출력 425MW → GT 310MW / CC 540MW로 수정
5. Long Ridge 사례를 "시연 완료"로 표현 수정
6. LCOE 등 경제성 수치는 전문가 검토 후 확정
