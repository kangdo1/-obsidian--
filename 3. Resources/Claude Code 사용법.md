# Claude Code 사용법

> 2026년 2월 3일 퇴직준비 세미나 1회차에서 배운 내용 정리

## Claude Code란?
터미널에서 Claude AI와 대화하며 작업할 수 있는 CLI 도구입니다.

## 기본 명령어

### 실행
```bash
claude                  # Claude Code 시작
```

### 슬래시 커맨드
| 명령어 | 설명 |
|--------|------|
| `/help` | 도움말 보기 |
| `/clear` | 대화 내용 초기화 |
| `/cost` | 토큰 사용량 확인 |
| `/quit` 또는 `Ctrl+C` | 종료 |

## 오늘 실습한 내용

### 1. 폴더 구조 확인
```
이 폴더의 구조를 설명해줘
```

### 2. 폴더 생성
```
PARA 폴더 구조 만들어줘
```

### 3. 파일 생성
```
Resources 폴더에 "Claude Code 사용법.md" 파일을 만들어줘
```

### 4. 디렉토리 이동
```
cd [폴더 경로]
```

## 자연어로 요청하기
Claude Code는 자연어를 이해하므로 일상적인 말로 요청하면 됩니다.

**예시:**
- "이 파일 읽어줘"
- "새 폴더 만들어줘"
- "이 코드가 뭐하는 건지 설명해줘"
- "git commit 해줘"
